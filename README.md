# Sales Performance Dashboard — Power BI

Dashboard de análisis de ventas retail para el mercado argentino,
desarrollado como proyecto final del curso de Power BI.

## Descripción

Análisis de datos de ventas correspondientes al período 2018–2020,
con foco en rentabilidad, costos y comportamiento por ciudad,
zona geográfica y forma de pago.

## KPIs principales

- Ventas totales: $730M
- Costos totales: $148M
- Rentabilidad promedio: 0.80
- Unidades vendidas: 53K

## Visualizaciones incluidas

- Ventas y rentabilidad por mes
- Ventas por forma de pago (efectivo vs tarjeta)
- Costos por tamaño de producto
- Rentabilidad por ciudad
- Mapa geográfico por latitud/longitud
- Análisis de entregas en tiempo vs vencidas
- Segmentación por zona (Sur, Norte, Occidente, Oriente)

## Herramientas

- Power BI Desktop
- DAX para métricas calculadas
- Filtros dinámicos por fecha, ciudad y forma de pago
